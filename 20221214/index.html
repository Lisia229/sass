01.Sass介紹

Sass 是 css　的預處理器，目的在減少　css 的重工

Sass 有變數、函式、流程控制、嵌套、繼承、混入mixins 這些功能讓我們使用，所以寫樣式時比 css 更有彈性。

Sass 可以更有效率維護樣式的結構

Sass 需要編譯成原生的 css 才能使用，因為瀏覽器看不懂 sass。

02. Sass 資料型態

  1. 1. number(數字):數字有分有單位或無單位，1,10,100,10px,10rem
  2. 2. string(字串):有分有引號字串與無引號字串
  3. 3. color(顏色):blue, #007bff, #fff, rgba(0,0,0,0.2)
  4. 4. boolean(布林值)
  5. 5. null(空值)
  6. 6. list(像陣列):用半形空格或逗號作為分隔，例如20px 0 20px
  7. 7. map(物件)

03. 變數

宣告使用$ ，變數位置即是作用域，就像 JS 的 let 宣告，寫在大括號裡面的是區域變數，大括號外的是全域變數

變數底線 _ 和連結線 - 兩者是一樣的變數，例如 $bg_dark、$bg-dark

使用變數可以維持網站的一致性，節省修改成本，只需要修改變數，不需要再 css 修改

04.流程控制

@if、@else if、@else

判斷條件可以小括號，也可以不括號

使用 and、or、not 處理判斷運算式的結果是 true 或 false

歐拉

@for 迴圈有兩種 through 與 to

@for $i from (start num) through (end num){...}，包含end num

@for $i from (start num) to (end num){...}，不包含end num
